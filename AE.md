# AE

## 了解面板

### 组织项目

按住 shift 多项选择文件。

创建归类文件夹：多选文件后，拖拽到工程面板底部文件夹图标上释放按键。

已选中素材文件格式创建合成：选择文件后，拖拽到工程面板底部合成图标上释放按键。

删除选中文件：选择文件后，拖拽到工程面板底部垃圾桶图标上释放按键。

批量改变标签颜色：多选文件后，再点击标签色彩。

双击工程面板名称可放大面板。

在工程面板属性栏的 Name、 Type 后空白处鼠标右键可以添加更多属性。

### 新建合成

第一种方法：顶部菜单栏直接新建合成。

第二种方法：将素材文件直接拖拽到首页新建合成的按钮上，会创建一个和素材文件格式匹配的合成。

改变已经创建合成的属性：在工程面板选中合成，顶部菜单栏选择合成中的合成属性设置。或者选中合成后快捷键 ctrl + k 。

新建层：先激活时间线窗口，然后顶部菜单栏选择层，新建。

在合成设置中可以修改合成的持续时间。例如合成的底层是一个视频，无视视频的时长，可以通过修改合成时长来让视频播放指定时间长度（当然也可以在时间线中修改层的持续时长）。

### 基本快捷键

V：选择

W：旋转

A：锚点

P：位置

S：缩放

R：旋转

T：不透明度

H：拖动

Z：放大缩小，逗号和句号也可以

按 shift 缩放是等比缩放。

## 使用合成

### 关键帧动画

变换器中各项变换属性名称左侧的表图标点击添加关键帧，然后修改各项参数。

### 导入带层的 PS 或 AI 素材

导入时 import as 选择 composition - Retain Layer Sizes ，控制点会匹配素材内容紧密排列，否则会和整体匹配。

### 添加效果

选中层，顶部菜单栏选效果，在效果控件中修改参数，右侧预览面板预览效果。如果知道效果的具体名称，也可在右侧效果&预设面板中的搜索框中直接搜索，然后将效果拖动到合成面板中的目标素材上。

U：选中层，按 U 显示所有的动画关键帧。

## 创建文本，设置文本动画

### 添加文本到合成

工作区调整为 Text ，右侧会显示文本相关的工具面板。

顶部工具栏选择文本工具，在合成窗口目标位置点击左键创建文本。

Alt + Enter：在文本框中换行。

新建的文本并非属于资产，而是创建在工程中的一个层。

### 使用文本预设对文本添加动画

第一种办法：先选中文本层，顶部菜单栏动画中选择浏览预设，通过 Bridge 进入 AE 自带的预设，找到目标后可以预览效果，确认后双击预设，自动跳回 AE 界面并将预设应用到选中的文本层上。此时如果动画应用成功，选中的文本会消失但控制点依然存在。

第二种办法：先选中文本层。和效果相同，可在效果&预设面板搜索名称，确认后直接双击。此时如果动画应用成功，选中的文本在合成中会消失但控制点依然存在。

按 U 打开动画属性，J、K 在所有可见关键帧之间左右横跳。

### 逐字母文本动画

选中层后，按 I 时间指示器会跳到该层的开头，然后展开文本层，在 text 右侧的动画箭头上选择不透明度，调为 0%，然后在动画器里调整不透明度的开始和结束的参数比例。

### 使用形状图层体现视觉层次结构

在该案例中，使用矩形画框，填充白色，然后调整位置将最顶层的图案覆盖，最后调整层顺序达到分离同色覆盖区域的目的。

这其中比较重要的就是要区分动画器中的变形器和层变形器的区别，类似于全局和局部的关系。

## 修复视频

### 稳定镜头

顶部菜单栏动画选项中的 Wrap Stabilizer VFX 效果，自动分析选中的视频片段，稳定抖动的镜头。

丝滑比例越高，出像素模糊即失真的范围也会越大。

方式中的 Subspace Warp 是指子空间变形，一个像素一个像素分析，稳定每一个像素。选择其他方式，失真的现象会更加严重，但适用于其他的目的需求。

自动缩放的比例低于 120% 是最好的，否则图像会变得有点软。

### 修复视频曝光

在将视频素材风格化之前（可以理解为调色之前），最好先保证画面是最好的质量，比如要矫正视频中的曝光问题，无论是曝光过度还是曝光不足。

效果&预设面板搜索 lumetri ，在 Basic Correction 中的 Tone 中，Exposure 曝光度 调整所有的像素；Contrast 对比度 调整最亮部分和最暗部分的差异；Highlights 高亮 Shadows 阴影 分别调整画面中的高亮度和阴影区域；Whites 绝对亮 Blacks 绝对暗 分别调整画面中包括的绝对亮和暗的区域（注意区分上面的高亮和阴影）。基础调整可以选择 Auto 自动调整。要在整段上保持自动设置的调整，需要借助关键帧。

### 增强视频色彩

效果 Lumetri Color 控件，White Balance 白平衡，Temperature 色温，Tint 色调。

一般应该先进行白平衡设置，然后再调节色温和色调。没有进行过白平衡校准，调节色调其实就是调节品红像素和绿色像素。

调色不要连续工作，眼睛会疲劳，短时休息会有新的感官！

## 创建自定义转场

### 使用蒙板自定义过渡

选中要作为蒙板的层，然后选择矩形工具或者钢笔工具，然后在合成面板的画面外侧圈画出蒙板的轮廓，按 M 打开蒙板属性，打关键帧，然后就跟关键帧动画一个原理了，运动打帧。另外点选 Mask 属性名时会批量选择到打的关键帧，便于复制应用。

### 使用形状图层构建图形过渡

选择已有的形状 AI 层，然后右键 create shapes from vector layer 创建新的形状图层，可以观察到原图层和新图层的图标是不同的，然后打开新的图层折叠属性，contents 右侧 add 图标，选择 repeater 重复器，参数中可以看到 copies 自动 3 个，说明该图层拷贝了 3 份，增多拷贝数，在合成面板中可以看到在原来图形的右侧有重影，然后调整 offset 偏移量，在偏移出合成窗口时打关键帧，然后偏移到合成窗口的另一侧再打关键帧。调整 start opacity 和 end opacity 来达到半透明拖影效果。

将要凸显的图标图层和原形状图层，在合适的中间位置（一般是两个关键帧中间），把这两层起点设置到该位置，快捷键是，ctrl 多选两层，按住 alt + [ ，自动对齐到时间指示针的位置。

### 使用图层创建透明度蒙板

首先介绍：合成面板下方一排按钮有红绿蓝三色的图标，表示的是图像的三色通道，还有 alpha 通道，alpha 通道表示的是透明度。

调整成 alpha 通道后，把不透明度调整为 50% ，再恢复成 RGB 通道，把背景图层拖拽出来铺满合成面板，然后，把已加好动画效果的字图层调整为可见，并在合成面板中显示出来。此时点击时间线窗口底部的 toggle switches\ modes 按钮，图层右侧进入模式 \ 蒙板栏，在背景图层的跟踪蒙板选项中选择 跟踪蒙板而不是亮度蒙板，这里会自动对应到背景图层的上一层，因为上一层是文字动画，在 AE 中会为文字等元素自动创建 alpha 通道，也就是蒙板的来源之一。再在背景图层应用一个 fast box blur 快速线框模糊，就能得到透明度蒙板了。

## 创建视觉效果（跟踪）

### 将类型放入视频素材中

样例：机舱内拍摄的飞机滑行镜头，在跑道上制造仿真人眼视觉变化的数字标志。

这里的类型实际指的是镜头跟踪，效果控件叫 3D Camera Tracker，3D 镜头跟踪，就是跟随摄像头移动，仿真视觉产生远近变化的效果。

先选中运动镜头素材，在顶部菜单栏 动画 选择 Track Camera ，合成面板的素材中会出现很多 x 点，鼠标移动会出现一个圆形目标区域，移动到目标区域后左键，按住 shift 加选 x 点使得目标区域范围更加广（这个无其实所谓），选择状态下圆形的目标区域其实是可移动的。选定后在圆形目标区的正中间右键，Set ground plane and origion，这一步是确定后面要置入元素的基本区域，然后继续右键 Creat Text and Camera，新建了一个文字层和摄像头，时间线中选中文字层编辑要输入的文字，在右侧的属性栏中更改字体属性，在层的基本属性中改变位置、旋转等参数。

### 模糊不需要的元素

样例：模糊一辆行驶中小汽车的车牌。

选中运动镜头素材，使用钢笔工具圈出车牌，这个过程中有需要精细操作的地方，把合成面板放大，圈画好后，在时间线窗口选中素材层按 M 调出 Mask ，右键选择 Track Mask 创建跟踪蒙板，时间线面板右侧会出现跟踪器面板，并把 Mask1 右侧有小箭头的框选为 None；在跟踪器中，方法选择 Position Scale & Rotation ，然后按面板上的播放键开始分析，分析过程中紧盯蒙板框，圈到非目标区域时按下播放键暂停，重新调整蒙板框，按 V 选择状态，蒙板框区域外点击一下鼠标出现蒙板框的控制点，然后拖拽调整，调整好后继续分析直到结束。

上一步在车牌上套了一个跟踪蒙板，下一步给跟踪蒙板添加模糊效果。

选中素材层，顶部菜单栏 效果 选择 快速方框模糊。在素材层中出现效果属性，将模糊半径调整到合适，此时发现是整个素材模糊，是因为还未将效果套用到蒙板上。在 Mask 1 右侧的箭头多选框中，将 None 改为 Add，None 状态的蒙板其实是还没有实际应用到素材上的。然后在效果属性中的 Compositing Options 合成选项旁边点击 + 号，引用了上面创建的蒙板，这时蒙板有了模糊效果，再调整 Mask1 中 Mask Feather 蒙板羽化到合适参数，跟踪蒙版模糊效果就有了。
